# Codebase Structure

```
OpenGram/
  App/            — Entry point: main.swift, AppDelegate.swift, OpenGramApp.swift
  CheckEngine/    — GrammarCheckerProtocol, HarperService, LLMService, CheckOrchestrator,
                    Suggestion, LLMConfig, LLMPrompts, DictionaryStore, LLMProviderProtocol
  SuggestionUI/   — OverlayWindow, OverlayController, UnderlineView, SuggestionPopoverPanel,
                    PopoverView, SuggestionDiffEngine, BoundsValidator, AXCallWatchdog,
                    TargetAppObserver, LLMSettingsView
  Shell/          — Menu bar shell
  TextEngine/     — AX text reading/writing
  TextMonitor/    — Real-time text change monitoring
  Hotkey/         — CGEventTap hotkey handling (Ctrl+Shift+G)
  Permissions/    — AX permission checks
  AppQuirks/      — Per-app compatibility workarounds
  Generated/      — HarperBridge.swift (auto-generated by build-harper.sh, do not edit)

OpenGramTests/    — Unit tests (Swift Testing framework)
OpenGramEntry/    — App entry target
harper-bridge/    — Rust crate wrapping harper-core, exposes UniFFI bindings
HarperBridge.xcframework — Pre-built xcframework (output of build-harper.sh)
scripts/test.sh   — SPM test runner
build-harper.sh   — Builds Rust crate → xcframework + Swift bindings
```
