<objective>
Rewrite OverlayController's three core paths: (1) replace raw boundsForRange with the BoundsValidator pipeline from Plan 01, supporting multi-line entries, (2) replace full-AXValue-overwrite accept with range-targeted AX write (D-11) plus proper re-query after accept (D-12), and (3) strip Tab/Enter key monitors keeping only Escape (D-14/D-15). Output: Modified OverlayController with validated bounds, range-targeted accept, and Escape-only key monitoring. Updated tests covering the new behavior.

        guard let view = underlineView,

        var newEntries: [UnderlineEntry] = []
        var survivingSuggestions: [Suggestion] = []
        var survivingOffsets: [(scalarStart: Int, scalarLength: Int)] = []

        for (i, suggestion) in suggestions.enumerated() {
            guard let rects = boundsValidator.validatedBoundsForRange(
